По вёрстке требуется:  
- Ручная вёрстка на bootstrap (не на конструкторах) 
- Адаптивная вёрстка под ПК, планшет, телефон 
- Оптимизация для Google PageSpeed 
- Анимация появления элементов при скроллинге страницы 
- Размещение на хостинге 
- Необходима передача заявок в AmoCRM + CMS с возможностью настраивать цели для Яндекс.Метрики и Google Analytics 
Вижу 2 варианта: 
А) Настроить прямую передачу данных из заявок по API в AmoCRM, а сайт повесить на WordPress 
Б) Повесить сайт на Drupal и связать формы с AmoCRM с помощью виджета Drupal, встроенного в AmoCRM (сможем и сами сделать, вопрос цены). 

Нюансы. 
- На главном экране фоновая картинка – слайдер из 3-х фото. 
- На 2-м экране пошаговая форма с формой обратной связи на последнем шаге. Надо сделать, чтоб отправлялись выбранные варианты со всех шагов.
